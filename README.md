# How to use Docker container

## Start container
open a terminal
run: docker-compose up


## Stop container
run: docker-compose stop


# URLs
your project: http://localhost
phpMyAdmin: http://localhost:8080
phpPgAdmin: http://localhost:8088/phppgadmin
